Stack:

Frontend:
1. Next JS
2. Shadcn
3. Axios
4. React-Hook-form
5. tailwind CSS

BackEnd:
1. Nest JS
2. Prisma (PostgreSQL)

How to start?
1. Copy project
2. Install dependencies
3. Start netflix-back-end by "npm run start:dev"
4. Start netflix-next by "npm run dev"
